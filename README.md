# Hanaloop Demo

Next.js 16, React 19, TypeScript, and Tailwind CSS 4 based demo app.

## Development

```bash
npm run dev
```

Open http://localhost:3000.

## Scripts

```bash
npm run lint
npm run build
```

## Project Structure

- `src/app/layout.tsx`: root document layout and metadata
- `src/app/page.tsx`: home route
- `src/app/globals.css`: global Tailwind and theme styles
- `public/`: static assets
